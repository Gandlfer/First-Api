# First-API

OS: Ubuntu <br>

## Features
- csgoskins
- copypasta

## .env template
```
MONGODBIP= <Hostname or IP of MongoDB server>
MONGODBPORT= <Port for the MongoDB>
PORT= <Port for API to run>
USERID= <Username for MongoDB authentication>
PASSWORD= <Password for MongoDB authentication>
```
